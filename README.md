# 📱 Simple Calculator App

## 📖 Description
The **Simple Calculator App** is an Android application built using **Kotlin** in **Android Studio**. The app performs basic arithmetic operations including **addition, subtraction, multiplication, and division**.

The application provides a simple and user-friendly interface where users can enter two numbers and select an operation to calculate the result.

---

## ✨ Features
- ➕ Addition of two numbers  
- ➖ Subtraction of two numbers  
- ✖ Multiplication of two numbers  
- ➗ Division of two numbers  
- ⚠️ Error handling for **division by zero**  

---

## 🛠 Technologies Used
- Kotlin
- Android Studio
- Android SDK
